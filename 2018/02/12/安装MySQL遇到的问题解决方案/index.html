Max OS X系统安装MySQL遇到的一些问题.

发表于 2018-02-12 | 分类于 Python
下载地址：MySQL 官网 下载安装包
第一次安装成功后无法启动Mysql，最终卸载重装ok，卸载命令：

打开终端输入以下命令:

  • sudo rm /usr/local/mysql

  • sudo rm -rf /usr/local/mysql*

  • sudo rm -rf /Library/StartupItems/MySQLCOM

  • sudo rm -rf /Library/PreferencePanes/My*

  • sudo rm -rf /Library/Receipts/mysql*

  • sudo rm -rf /Library/Receipts/MySQL*

  • sudo rm -rf /var/db/receipts/com.mysql.*

此时我们在终端输入此时我们在命令行输入mysql -uroot -p命令会提示没有commod not found，我们还需要将mysql加入系统环境变量:
  • cd /usr/local/mysql/bin
  • vim ~/.bash_profile
  • 添加PATH=$PATH:/usr/local/mysql/bin
  • 保存退出（按esc,然后输入’:’,再输入wq,回车保存并退出）
  • source ~/.bash_profile
打开终端,尝试登陆mysql -uroot -p出现以下错误：
ERROR 1045 (28000): Access denied for user ‘username’@’localhost’ (using password: YES)问题解决办法

打开终端输入以下命令:

  • sudo service mysqld stop
  • cd /usr/local/mysql/bin
  • sudo mysqld_safe –skip-grant-tables

接下来打开一个新的终端输入以下命令：

  • mysql
  • use mysql;

  • see Note1 below for next line.

  • UPDATE user SET password=PASSWORD(‘你的密码’) WHERE user = ‘root’;

  • exit;

最后在终端执行：

  • sudo service mysqld start
再次登陆mysql -uroot -p输入密码就会出现：

Welcome to the MySQL monitor. Commands end with ; or \g.
Your MySQL connection id is 5
Server version: 5.7.21 MySQL Community Server (GPL)

Type ‘help;’ or ‘\h’ for help. Type ‘\c’ to clear the current input statement.

mysql>

好了，MySQL的基本配置就完了。
