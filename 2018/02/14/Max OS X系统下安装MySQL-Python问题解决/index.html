Max OS X系统下安装MySQL-Python问题解决

发表于 2018-02-14 | 分类于 Python

终端执行pip install MySQL-Python报错：

Command "python setup.py egg_info" failed with error code 1 in /private/tmp/pip-build-q9tj25am/MySQL-Python/

<– !more –>

以下是解决方案，依次操作即可成功安装MySQL-Python

1.下载安装包:

MySQL for Python官网下载地址

2.解压刚下载的安装包,cd 到其目录下再执行pip install MySQL-Python报如下错误:

>
Downloading http://pypi.python.org/packages/source/d/distribute/distribute-0.6.28.tar.gz
　………….

　　File “/System/Library/Frameworks/Python.framework/Versions/2.7/lib/python2.7/urllib2.py”, line 558, in http_error_default raise HTTPError(req.get_full_url(), code, msg, hdrs, fp)

　urllib2.HTTPError: HTTP Error 403: SSL is required

　 　

为了解决这个错误，直接将http://pypi.python.org/packages/source/d/distribute/distribute-0.6.28.tar.gz下载到当前文件夹。先暂时不执行pip install MySQL-Python命令

3.修改site.cfg文件，里面指明了mysql_config的路径：

是mysql没有安装导致路径中没有，因此需要安装mysql，同时，如果有需要，要修改site.cfg文件，里面指明了mysql_config的路径。

打开site.cfg文件，修改mysql_config = /usr/local/mysql/bin/mysql_config保存并退出

4.再执行pip install MySQL-Python就可以成功安装MySQL-Python

安装了mysql后，问题解决。

在python命令行下 import MySQLdb成功，说明OK了。
